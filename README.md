## ðŸ“Œ Overview

**QuynhLe-Portfolio** is a personal portfolio backend project that enables seamless user engagement through:
- Handling HTTP requests using **Node.js**
- Managing cross-origin access using **CORS**
- Supporting email contact forms via **Gmail SMTP (Nodemailer)**

This project provides the backend for a developer portfolio and can be extended with a frontend or deployed with services like **Render** or **Railway**.
